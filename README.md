# AI-Powered KYC Verification System

An intelligent identity verification system that automates KYC validation using OCR, face matching, liveness detection, deepfake detection, and risk analysis.

## Features

- OCR-Based Document Verification
- Face Matching & Identity Validation
- Liveness Detection
- Deepfake Detection
- Risk Scoring Engine
- Automated Decision Making
- User-Friendly Interface

## Tech Stack

- Python
- OpenCV
- EasyOCR / Tesseract OCR
- InsightFace
- NumPy
- Deep Learning Models

## Setup Instructions

### Prerequisites

- Python 3.10+
- pip

### Installation

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python main.py
```
